# AI Study Buddy - Demo Scenarios

This guide provides realistic scenarios to showcase the Study Buddy's capabilities.

## Scenario 1: Math Homework Help

### Setup
1. Open a math textbook or printed homework to a page with equations
2. Start Study Buddy in **🧮 Solve Step-by-Step** mode
3. Point camera at a specific problem

### Conversation Flow

**You**: [Point camera at problem]  
**Action**: Click "Analyze Image"  
**AI**: "I can see a quadratic equation: x² + 5x + 6 = 0. This appears to be a factoring problem."

**You**: [Voice] "How do I solve this?"  
**AI**: "Great question! Let's think about this step by step. First, can you tell me what two numbers multiply to give us 6 and add up to give us 5?"

**You**: [Voice] "Um, 2 and 3?"  
**AI**: "Excellent! You're exactly right. So we can factor this as (x + 2)(x + 3) = 0. Now, what happens when we set each factor equal to zero?"

**You**: [Voice] "x equals negative 2 or negative 3"  
**AI**: "Perfect! You've got it. The solutions are x = -2 and x = -3. You can verify by substituting these back into the original equation."

### Key Feature Demonstrated
- Problem identification through vision
- Socratic method teaching (asking guiding questions)
- Step-by-step guidance without giving direct answers
- Multi-turn conversation maintaining problem context

## Scenario 3: Language Learning

### Setup
1. Open a book or display text in a foreign language
2. Start Study Buddy in **📖 Explain** mode
3. Point camera at the text

### Conversation Flow

**You**: [Point camera at Spanish text]  
**Action**: Click "Analyze Image"  
**AI**: "I can see Spanish text that reads 'El gato está en la casa.' This is a simple sentence structure."

**You**: [Voice] "Can you translate and explain the grammar?"  
**AI**: "Of course! It translates to 'The cat is in the house.' Let's break it down: 'El gato' is the subject meaning 'the cat,' 'está' is the verb 'to be' conjugated for third person, and 'en la casa' means 'in the house.' The verb 'estar' is used here for location."

**You**: [Voice] "Why not 'ser' instead of 'estar'?"  
**AI**: "Excellent question! Spanish has two verbs for 'to be.' We use 'estar' for temporary states and locations, while 'ser' is for permanent characteristics. Since the cat's location is temporary, we use 'está' from the verb 'estar.'"

### Key Feature Demonstrated
- Text recognition in foreign languages
- Detailed grammatical explanations
- Context-aware follow-up answers
- Educational breakdown of concepts

## Scenario 4: Chemistry Formula Review

### Setup
1. Display or write chemical formulas/reactions
2. Start Study Buddy in **📖 Explain** mode
3. Point camera at the formulas

### Conversation Flow

**You**: [Point camera at: 2H₂ + O₂ → 2H₂O]  
**Action**: Click "Analyze Image"  
**AI**: "I see a chemical equation showing two molecules of hydrogen gas reacting with one molecule of oxygen gas to produce two molecules of water."

**You**: [Voice] "Why do we need two water molecules on the right side?"  
**AI**: "That's all about balancing the equation! On the left, we have 4 hydrogen atoms (2 in each H₂ molecule) and 2 oxygen atoms (from O₂). To balance, we need 2 water molecules on the right, giving us 4 hydrogens and 2 oxygens total. This follows the law of conservation of mass."

**You**: [Voice] "What type of reaction is this?"  
**AI**: "This is a synthesis reaction, also called a combination reaction, where simpler substances combine to form a more complex compound. It's also a combustion reaction when hydrogen burns in oxygen, releasing energy."

### Key Feature Demonstrated
- Chemical notation recognition
- Scientific explanation with reasoning
- Multi-concept answers (balancing + reaction types)
- Clear educational communication

## Scenario 6: Programming Code Review

### Setup
1. Display code on screen or in a textbook
2. Start Study Buddy in **📖 Explain** mode
3. Point camera at the code

### Conversation Flow

**You**: [Point camera at Python function]  
**Action**: Click "Analyze Image"  
**AI**: "I see a Python function that appears to implement a bubble sort algorithm. The function takes a list as input and uses nested loops to sort the elements."

**You**: [Voice] "Can you explain how bubble sort works?"  
**AI**: "Sure! Bubble sort compares adjacent elements and swaps them if they're in the wrong order. It repeats this process, 'bubbling' larger values to the end of the list. In the code you're showing, the outer loop runs n times, and the inner loop does the comparisons and swaps."

**You**: [Voice] "What's the time complexity?"  
**AI**: "Bubble sort has a time complexity of O(n squared) because of the nested loops. Each element is compared with every other element. This makes it inefficient for large datasets compared to algorithms like quicksort or mergesort, which are O(n log n)."

### Key Feature Demonstrated
- Code recognition and understanding

## Tips for Best Experience

### Camera Setup
- **Lighting**: Ensure good lighting on study material
- **Distance**: Hold phone/position laptop 6-12 inches from material
- **Angle**: Point camera directly at text (not at steep angle)
- **Stability**: Keep camera steady during analysis
- **Focus**: Make sure text is in focus and readable

### Voice Interaction
- **Clarity**: Speak clearly and at normal pace
- **Natural**: Ask questions as you would to a tutor
- **Pauses**: Brief pause after asking lets VAD detect end
- **Follow-ups**: Don't restart listening - it auto-resumes after AI speaks
- **Volume**: Normal speaking volume works; no need to shout

### Study Material Tips
- **Contrast**: Dark text on white paper works best
- **Size**: Larger text is easier for VLM to read
- **Focus**: Point at specific sections, not entire page
- **Handwriting**: Write clearly if using handwritten notes
- **Diagrams**: Well-labeled diagrams work great

### Mode Selection Guide
- **Explain Mode**: First time learning something new, need detailed explanations
- **Quiz Mode**: Testing yourself, preparing for exams, active recall practice
- **Solve Mode**: Working through problems, homework help, step-by-step guidance

## Troubleshooting Common Scenarios

### "AI can't see the text clearly"
- Improve lighting
- Move camera closer
- Ensure text is in focus
- Try reanalyzing from better angle

### "Voice not picking up my questions"
- Check microphone permissions
- Speak a bit louder
- Ensure you're in "Listening" state
- Wait for audio level indicator

### "AI responses seem generic"
- Make sure you analyzed the image first
- Ask specific questions about visible content
- Reference specific parts: "What about the equation at the top?"

### "Conversation loses context"
- Keep same study material in frame
- Reanalyze image if switching to new content
- Stay in same session for related questions
